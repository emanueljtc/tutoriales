Pseudoclases

Estilos especiales dependiendo del contexto, posicion o estado

:root

La Pseudo-clase :root selecciona el elemento raiz de un documento aplicado a HTML, :root representa el elemento <html> y es identico al selector html, exepto que su nivel de especificidad es mayor. Sirve para declarar variables CSS.

:hover

La Pseudo-clase CSS :hover se representa el usuario coloca el puntero sobre algún elemento, pero no necesariamente está activo. En pantallas touch :hover es problemático.

:active

La pseudo-clase :active cubre el caso en el que el elemento esta siendo activado por el usuario. Cuando se interactua con el raton en el momento en el que el usuario presiona el raton hasta que lo suelta. A menudo es usado en los elementos <a> y <button> de HTML.

El orden si importa:

  1. :link {...}
  2. :visited {...}
  3. :hover {...}
  4. :active {...}

:target

La peudo-clase :target representa el elemento único, si existe alguno, con id coincidente con el identificador de fragmentos de la URL del documento.

:nth-child :first-child :last-child

  • nth-child: selecciona el elemento indicado que sea el enésimo hijo de su elemento padre.
  • first-child: selecciona el elemento indicado que sea el primer hijo de su elemento padre.
  • last-child: selecciona el elemento indicado que sea el ultimo hijo de su elemento padre.

:nth-of-type :first-of-type :last-of-type

  • nth-of-type: selecciona el elemento indicado que sea el enesimo hijo en su tipo sin importar la jerarquia dentro del elemeto padre.
  • first-of-type: selecciona el elemento indicado que sea el primer hijo en su tipo sin importar la jerarquia dento del elemento padre.
  • last-of-type: selecciona el elemento indicado que sea el ultimo hijo en su tipo sin importar la jerarquia dentro del elemento padre.
